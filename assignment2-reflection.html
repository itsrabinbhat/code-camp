-- Introduction --

In crating the website for our event, I began to focus on the design of homepage that would immediately grab viewer's attention. The inclusion of the event's name, logo, and tagline required thoughtful consideration to create a visually appealing header and providing the necessary information. The nav bar is designed so that it stands out in the website and is easily readable. Additionally, I have maintained color and design consistency throughout the website, which makes the website visually cohesive and provides a seamless user experience.

-- Challenges --

Throughout the project, I faced several challenges that tested my problem-solving skills and technical knowledge. Here are some of them:

  • Design Consistency:

    Maintaining a consistent aesthetic throughout the various sections of the webpage, while ensuring each had its unique appeal, required careful planning and execution.

  • Interactive Elements:

    Implementing interactive CSS features such as hover effects and transitions demanded a deeper understanding of CSS

  • Layout Techniques:

    Positioning elements using CSS properties like float, position, and z-index presented a learning curve in understanding how elements stack and align in a multi-layered layout.

-- Learnings --

The challenges I faced led to a series of learnings that have enriched my skill set as a student of Computer Systems Technology:

  • HTML5 and Semantic Elements:

    I learned to effectively structure content using semantic HTML, improving the accessibility and SEO friendliness of the webpage.

  • CSS3 Advanced Techniques:

    Through the use of Google Web Fonts, gradients, and responsive design principles, I enhanced the visual appeal and user experience of the website.

  • Debugging and Testing:

    I developed better debugging skills while testing interactive elements and layouts, learning to use browser developer tools more effectively.

-- Summary --

In conclusion, this project was not just about building a webpage, but also about growth and expanding my knowledge of web development as a Computer Systems Technology. The difficulties I encountered pushed me to explore new technologies and techniques, resulting in a product that not only meets the design requirements but also reflects my dedication to learning and improvement.
